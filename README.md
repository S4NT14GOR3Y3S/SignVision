# SignVision – Reconocimiento de Lenguaje de Señas con IA

## Descripción

SignVision es una aplicación desarrollada en Python que utiliza visión por computadora e inteligencia artificial para reconocer gestos del lenguaje de señas usando una cámara web.

## Tecnologías utilizadas

- Python
- OpenCV
- TensorFlow / Keras
- NumPy

## Requisitos

- Python 3.8 o superior
- Cámara web

## Instalación

1. Clonar el repositorio

git clone https://github.com/tuusuario/signvision.git

2. Instalar dependencias

pip install opencv-python
pip install numpy
pip install tensorflow
pip install keras

## Uso del sistema

### 1 Capturar gestos

python create_gesture_data.py

### 2 Entrenar modelo

python DataFlair_trainCNN.py

### 3 Ejecutar reconocimiento

python model_for_gesture.py
